# 📝 Reusable Footer Component

Un composant footer réutilisable construit avec React.  
Il présente plusieurs sections de navigation et un texte de copyright, avec une mise en page responsive réalisée en CSS.

## 🚀 Fonctionnalités

- Composant React fonctionnel réutilisable
- Organisation du footer en plusieurs sections de liens
- Titres dynamiques générés avec l'attribut `data-title`
- Design responsive avec Flexbox
- Chargement et rendu de React directement dans le navigateur via CDN


## ⚙️ Prérequis

- Un navigateur web moderne
- Une connexion internet pour charger React, ReactDOM et Babel via CDN


## 📂 Structure du projet

reusable-footer-component/  
│── index.html      # Structure principale et montage de l'application React  
│── styles.css      # Mise en forme et responsive du footer  
│── index.jsx       # Composant React Footer  
│── README.md       # Documentation  


## 🛠️ Technologies utilisées

React – Création du composant d’interface utilisateur  
JavaScript (JSX) – Logique et structure du composant  
HTML5 – Structure de la page  
CSS3 – Mise en forme et responsive design  
Babel Standalone – Compilation du JSX directement dans le navigateur  


## 🎯 Objectif pédagogique

Ce projet illustre la création d’un **composant React réutilisable**.  
Il permet de comprendre comment structurer une interface avec React, organiser des sections de navigation dans un footer et appliquer un style responsive avec CSS.

L'utilisation de **Babel Standalone et des CDN React** permet également de tester rapidement des composants React sans configuration d’environnement de développement.
